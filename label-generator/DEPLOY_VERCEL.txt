🔧 PASOS PARA DEPLOY EN VERCEL:

1. PREPARAR REPOSITORIO:
   ```bash
   # Si no tienes git inicializado
   git init
   git add .
   git commit -m "EFO Label Generator - Sistema completo"
   ```

2. SUBIR A GITHUB:
   - Crear repositorio en github.com
   - Conectar y subir código:
   ```bash
   git remote add origin https://github.com/tu-usuario/efo-label-generator.git
   git push -u origin main
   ```

3. DEPLOY EN VERCEL:
   - Ir a vercel.com
   - "New Project"
   - Importar desde GitHub
   - Seleccionar el repositorio
   - Framework: Next.js (auto-detectado)
   - Configuración automática ✅
   - Deploy!

4. VARIABLES DE ENTORNO (OPCIONAL):
   En Vercel > Settings > Environment Variables:
   ```
   NEXT_PUBLIC_APP_NAME=EFO Label Generator
   NEXT_PUBLIC_STORAGE_TYPE=local
   ```
